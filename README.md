# Bitcoin Price Prediction using Deep Learning

## Overview

This project analyzed Bitcoin historical price data to understand the factors influencing future price movements and built a Deep Learning model to predict Bitcoin prices accurately. Key historical price trends and sequential market patterns were used as dependent variables to generate future forecasts.

Bitcoin price data has been taken from Yahoo Finance.

The model uses:

- Historical Closing Price
- Time Series Data
- Sequential Price Patterns

The application predicts future Bitcoin prices.
